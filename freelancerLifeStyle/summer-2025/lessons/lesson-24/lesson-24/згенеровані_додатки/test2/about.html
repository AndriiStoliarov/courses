Про нас

Dog Gallery - це проект, створений для демонстрації можливостей сучасного веб-розробки з використанням модульної архітектури та компонентного підходу.

Технології

  • Vanilla JavaScript (ES6+ модулі)
  • HTML5 семантика
  • CSS3 з Flexbox та Grid
  • Dog CEO API для отримання зображень

Особливості

  • Модульна архітектура з перевикористанням компонентів
  • Адаптивний дизайн
  • SPA-подібна навігація
  • Асинхронне завантаження даних
